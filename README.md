# Burp-WSR

Burp-WSR is a minimal extension for Burp Suite that adds missing match and replace features for WebSockets.


## How to use

1. Copy `wsr-config.json` and `wsr-1.0-jar-with-dependencies.jar` to the folder where Burp Suite is located.
2. Modify `wsr-config.json` to replace your strings as needed.
3. Enable the extension in Burp's config.

## How to build

`mvn package`
